The Witch is a compact and versatile polyphonic synthesizer.

It comes with a set of eight factory patches, implementing four different synthesis engines:

WaveBank

Multidimensional wavetable synthesizer with x/y morphing. 
Polyphony: 4 voices
A: Frequency
B: Morph X
C: Morph Y
D: Envelope
E: Overdrive

Out G: LFO2 Agnesi
1: Note

Out 5: LFO1 Clock
Out 6: LFO2 Clock
In Left: FM Left
In Right: FM Right

The WaveBank patch loads two wav resources, each containing a bank of 8 by 8 single cycle waveforms. One is for the left output channel, the other for the right. The wavebank format is compatible with the popular WaveEdit software and its online library of free wavetable banks.
